A base de um HTML5

Para começar, o corpo do HTML5 é composto por:

        
 <!DOCTYPE html>
        <html lang="(Aqui você coloca a lingua desejada, no caso pt-br)">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Document</title>" 
        </p>
        </head>
        <body>
        </body>
        </html>

Um HTML é o conteúdo, textos, imagens, vídeos e tabelas

O CSS são as cores, formas, tamanhos e posições. Existe diferença entre forma e sentido. Forma seria como sublinhamos algo em HTML, usando a TAG <u> ou então em css, alterando a forma, e o sentido seria em HTML usando a tag <ins>, sendo assim um texto inserido
