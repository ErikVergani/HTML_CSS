Como adicionar imagem!

Buscando a imagem na pasta:

Caso você tenha criado uma pasta e adicionado a imagem dentro, você deverá buscala, basta colocar o nome da pasta/ e após isso o nome do arquivo. Se a imagem estiver em alguma pasta anterior, será necessário colocar ../ para que o algoritmo entenda que preciamos voltar uma pasta para a busca.

Adicionado imagens por link externo

Para adicionar uma imagem por link externo, será necessário utilizar o comando <img src="" alt="">, após isso basta adicionar o link da IMAGEM em src e um descritivo em alt caso a imagem não seja carregada.


Para imagens locais, o processo é o mesmo porém em src, você deverá colocar o nome da imagem e caso necessário a pasta
