Como utilizar links

como linkar um CSS, ou paginas HTML

Para linkar um arquivo CSS use: <link rel="stylesheet" href="style.css">

Para linkar paginas html use a tag: <a href=""></a> , use os mesmo passos para adição de imagens, pórem com seu arquivo HTML.

Linkando sites externos

Para linkar um site externo você precisar ao invés de colocar o arquivo html no href colocar https://www. e o link desejado
Exemplo:

clique <a href="https://www.google.com.br" rel="external" target="_blank">aqui</a> para ir ao google

Resultado:
Clique aqui para ir ao google.

Lembre-se de colocar target="_blank" para que seja aberta em uma nova guia(apenas para sites externos) e no rel external para sites externos ou self para site interno, como nossas páginas.

LISTAS!

Outra coisa muito importante em HTML são as listas. Existem dois tipos de listas em HTML, ordenadas e não ordenadas.

Para criar uma lista ordenada precisamos da tag <ol>, que significa ordered list. Após isso, utilizamos a tag <li> para adicionar os itens. todos li precisam estar dentro da ol

podemos personalizar diretamente na ol o tipo que quermos, basta dar um espace e adicinar type que logo nos mostrará os modelos disponíveis.

Para listas não ordenadas o processo é o mesmo porém a tag é ul, unordered list

Exemplo:

  1. Acordar
  2. ligar pc
  3. tomar café

podemos adiconar listas dentro de listas, misturando-as ou não. nas ordenadas caso precise adicionar algo no meio não haverá problemas pois o próprio código trocará a númeração.
